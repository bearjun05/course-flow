> 앞서 "폴더 위치로 구분"이라고 답변 주셨으나, 쓰다 만 상태라 재확인 요청.

- 정책: Drive에 직접 올리면 자동 감지
- 세부 명확화가 필요한 부분:
  - 자동 감지는 **폴더 위치**만으로 충분한가요? (01. 원본영상에 파일 올리면 촬영 완료, 03. 최종영상 + 04. 최종자막 올리면 편집·자막 완료 등)
  - 파일명 포맷(`N-N_(m)_요청사항` / `N-N_제목`)을 **꼭 지켜야** 자동 감지되나요? 포맷 안 지킨 파일은 무시?
  - `05. 요약노트` 폴더에 JSON 파일이 올라가면 어떤 이벤트가 트리거되나요? (승인 단계로 넘어가는 트리거? 아니면 단순 로그?)

**Q-12. "보완" 판정에 대한 추가 설명 요청**

> 원래 영상 검수 기능에 "승인 / 보완 / 재촬영" 3단계 판정이 있었는데, 지금은 검수 UI가 삭제되고 "검수완료" 체크 토글로 단순화됨.
> 그래서 "보완"이라는 개념이 지금 시스템에 없는 상태예요.

- 검수자가 체크 후 승인 단계로 넘어가는데, 영상에 문제가 있으면 어떻게 처리하나요?
  - A. 체크 자체를 안 누름 → 편집자/자막자와 별도로 슬랙에서 커뮤니케이션
  - B. "검수 반려" 같은 버튼이 있어야 하고, 누르면 자동으로 편집·자막 단계로 되돌려짐 + 슬랙 알림
- 이전 영상과 수정본을 비교하거나 재검수 이력을 남겨야 하나요?

→ 답변:

**Q-17 (재확인). 강의명 변경 기능**

> "강의명이 바뀌면 폴더 이름도 바뀌어야 해"라고 답변 주셨는데, 현재 **프론트에서 강의명을 수정할 수 있는 UI가 없음**.

- 강의명 수정 기능을 **추가**해야 하나요?
  - A. 네, 추가 필요 → 프로젝트 상세 헤더에서 편집 가능하게?
  - B. 아니요, 강의명 변경은 백오피스에서 하고 자동 동기화
- 강의명이 바뀌면: Drive 폴더명 / 슬랙 채널명 / 백오피스 전부 동기화가 맞나요?
