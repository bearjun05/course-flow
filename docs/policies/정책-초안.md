## 1. 강의 제작 프로세스 (단계별 전이 조건)

### ✅ 확정: 5단계 파이프라인

```
교안 → 촬영 → 편집.자막 → 검수 → 완료
```

| 단계 | 시작 조건 (트리거) | 종료 조건 | 담당자 |
|------|-------------------|-----------|--------|
| 교안 | 새 강의 생성 시 모든 장이 교안 단계로 시작 | 커기매가 노션 링크를 교안링크로 첨부 | 커기매 |
| 촬영 | 교안링크 첨부 완료 | 튜터가 구글 드라이브 원본영상 폴더에 파일 업로드 | 튜터 |
| 편집.자막 | 원본영상 업로드 감지 | 최종영상+최종자막 폴더에 파일 업로드 | 편집자 + 자막자 |
| 검수 | 최종파일 업로드 감지 | 검수자가 "검수완료" 버튼 클릭 | 검수자 |
| 완료 | 검수완료 클릭 (→ 승인 대기) | PM이 "승인" 버튼 클릭 | PM |

### ✅ 확정: 프로젝트(강의) 완료 조건

**모든 장이 "완료" 단계** 에 도달해야 프로젝트 완료 가능.

### ✅ 확정: 장 간 병렬 작업 가능

CH1 편집 중 CH2 촬영 동시 가능. 장마다 독립적 파이프라인.

### ✅ 확정: 상태 자동 전이

파일 업로드 감지 → 다음 단계 자동 이동 + 다음 담당자에게 슬랙 알림.
PM이 수동으로 상태를 바꿀 필요 없음 (검수완료/승인 제외).

## 2. 구글 드라이브 폴더 구조

### ✅ 확정: 폴더 트리

```
{루트 디렉토리}  ← 환경변수로 설정
└── {강의명}  ← 백오피스와 동일한 이름
    └── {강의명}_v{버전}
        └── {N장}
            ├── 01. 원본영상       ← 튜터가 업로드
            ├── 02. 편집 (vrew)    ← 편집자가 업로드
            ├── 03. 최종영상       ← 자막자가 업로드
            ├── 04. 최종자막       ← 자막자가 업로드
            └── 05. 요약노트       ← 자막자가 업로드 (JSON)
```

## 3. 프로젝트 상태 (7가지)

### ✅ 확정

| 상태 | 의미 | 대시보드 표시 |
|------|------|--------------|
| 기획 | 초기 기획. 챕터 0개/튜터 미정 가능 | "진행 중" 탭에만 표시. 메트릭/진척표 제외 |
| 교안 | 교안 작성 중 | 표시 |
| 촬영 | 촬영 중 | 표시 |
| 편집·검수 | 편집/자막/검수/승인 | 표시 |
| 롤아웃 | 출시 준비 | 표시 |
| 완료 | 제작 완료 | "전체" 탭에만 표시 |
| 중단 | 일시 중지/취소 | "전체" 탭에만 표시 |

### ✅ 확정: 대시보드 탭별 필터

- **"진행 중"**: 완료/중단 제외 (기획~롤아웃)
- **"전체"**: 모두 표시 (중단 포함)
- **요약 메트릭/진척표**: 기획/완료/중단 제외 (교안~롤아웃만)

## 4. 대시보드 (에듀옵스 `/`)

### ✅ 확정: 요약 메트릭

| 카드 | 값 | 설명 |
|------|-----|------|
| 오늘 태스크 | 현재 "진행" 상태인 전체 태스크 수 | 활성 프로젝트(교안~롤아웃) 기준, D-Day 가까운 순 |
| 지연 | 마감 초과 프로젝트 수 | D-Day < 0 이고 완료가 아닌 프로젝트 |

### ✅ 확정: 진척표 (도트 매트릭스)

- **5단계 컬럼**: 교안 / 촬영 / 편집.자막 / 검수 / 완료
- **도트 색상**: 연두 계열 가장 진한 색(#8AAE50) 일괄 통일
- **지연 도트**: 빨강(#F9919E)
- **5장 이상 연속 같은 단계**: 묶음 도트로 표시 (예: "1-5")
- **행 클릭 또는 도트 클릭**: 강의 상세 페이지로 이동

### ✅ 확정: 프로젝트 그리드

- **필터 기본값**: "전체 상태", "전체 사업부" (식별 가능한 라벨)
- **사업부**: KDT / KDC / 기타
- **숨기기 기능**: 제거됨

## 5. 역할

### ✅ 확정: 6가지 역할

| 역할 | 소속 | 주요 업무 |
|------|------|-----------|
| PM | 내부 | 전체 제작 관리, 승인 최종 확인 |
| 커기매 | 내부 | 커리큘럼 기획, 교안링크 첨부, 콘텐츠 검수 |
| 튜터 | 외부 | 강의 촬영, 원본영상 업로드 |
| 편집자 | 외부 | 영상 편집 (vrew) |
| 자막자 | 외부 | 최종영상 + 최종자막 + 요약노트 제작 |
| 검수자 | 외부 | 영상 품질/내용 검수, 검수완료 처리 |

### ✅ 확정: 시스템 접근

- **에듀옵스** (`/`): PM + 커기매 (내부)
- **에듀웍스** (`/eduworks`): 튜터 + 편집자 + 자막자 + 검수자 (외부)

---

## 6. D-Day와 신호등

### ✅ 확정

- D-Day = 롤아웃일 - 오늘 (달력일, 주말 포함)
- D-Day 색상: 마감초과=빨강, 3일이내=노랑, 그 외=회색
- 신호등: 마감 초과 시 자동 빨강. 노랑은 PM 수동 설정.
