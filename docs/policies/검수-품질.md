# 검수 및 품질 기준

> 영상 검수 프로세스와 판정 기준을 정의한다.

## 규칙

- **[RV-001]** 검수는 강(講) 단위로 진행 (1-1 = 1장 1강)
  - 배경: 강이 피드백/검수/판정의 최소 단위
  - 적용 화면: 프로젝트 상세 > 영상 피드백 탭

- **[RV-002]** PM 검수와 커기매 검수는 독립적으로 수행, 관점이 다름
  - PM: 영상 퀄리티 (음성 전달력, 화면 구성, 자막, 음향)
  - 커기매: 콘텐츠 정합성 (교안 일치, 학습 목표 달성, 난이도)
  - 적용 화면: 프로젝트 상세 > 영상 피드백 탭

- **[RV-003]** 판정 결과는 승인 / 보완 / 재촬영 3단계
  - 배경: 피드백 심각도에 따라 후속 액션이 달라짐
  - 적용 화면: 프로젝트 상세 > 영상 피드백 탭

## 데이터 구조

- 한 강에 복수 영상 파일 가능 (화살표로 이전/다음 이동, 자동 연속 재생)
- 검수 항목: 6문항 점수 + 코멘트
- 피드백 문구 자동 생성 → 슬랙 채널로 전달

## 미결정 사항

- [ ] 승인/보완/재촬영 판정의 구체적 기준 (점수 몇 점 이상이면 승인?)
- [ ] "보완"과 "재촬영"의 차이 기준은?
- [ ] PM과 커기매 판정이 다를 때 최종 결정 기준은?
- [ ] 재촬영 시 기존 피드백 이력은 어떻게 관리하는가?
