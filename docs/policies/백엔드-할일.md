- [ ] **프로젝트 생성 시 (리뉴얼)**:
  - 외부 API로 기존 강의 검색 결과 제공 (강의명 + 최신 버전)
  - Drive에서 기존 강의명 폴더 탐색
  - 폴더 있음 → `{강의명}_v{다음버전}/` 만 추가 생성
  - 폴더 없음 → 강의명 폴더부터 신규 생성
- [ ] **기획 완료 모달에서 장 입력 시**: `{N}장_{제목}/` 및 하위 5개 폴더 자동 생성
  - `01. 원본영상`, `02. 편집 (vrew)`, `03. 최종영상`, `04. 최종자막`, `05. 요약노트`
- [ ] **상세 페이지에서 장 추가 시**: 해당 장 폴더 자동 생성

### 파일 감지 → 자동 전이
- [ ] **파일 업로드 감지 시스템 방식 결정** (백엔드 영역)
  - Drive Push Notification (Watch API) vs 주기 폴링 중 선택
  - 감지 지연 허용 범위, 재시도 정책, 중복 감지 처리
- [ ] **파일명 포맷 파싱 + 검증 (정규식 설계는 백엔드 영역)**:
  - `N-N_(m)_요청사항` (원본영상): 장-강(1-1, 3-2), 분할 합본 순서(선택), 편집 요청사항 추출
  - `N-N_제목` (최종영상/최종자막): 장-강, 제목 추출
  - **포맷 위반 파일은 무시** (자동 감지 트리거 안 됨)
  - 포맷 위반 시 PM에게 경고 알림을 줄지 여부 결정 필요
- [ ] **자동 전이 트리거** (폴더 위치 + 파일명 포맷 둘 다 만족해야 작동):
  - 커기매가 교안 링크 등록 → "교안 → 촬영" 전이 + 촬영 완료 슬랙 알림
  - `01. 원본영상` 폴더에 강별 파일 업로드 감지 → "촬영 → 편집·자막" 전이 + 알림
  - `03. 최종영상` + `04. 최종자막` 모두 감지 → "편집·자막 → 검수" 전이 + 알림
  - `05. 요약노트` JSON: 단순 업로드 확인용 (전이 트리거 아님)
- [ ] **중복 업로드 처리 정책** (같은 파일명 재업로드, 덮어쓰기 vs 버전 보존)

## 🌐 API 엔드포인트 (예상)

### External (강의 검색)
- [ ] `GET /api/external/courses?q=` — 외부 시스템(백오피스 등)에서 기존 강의 검색
  - 응답: `[{ id, title, latestVersion }]`
  - 제작 요청 폼 리뉴얼 분기에서 사용

### Projects
- [ ] `GET /api/projects` — 프로젝트 목록 (필터, 페이지네이션)
- [ ] `GET /api/projects/:id` — 상세
- [ ] `POST /api/projects` — 제작 요청 폼 제출
- [ ] `PATCH /api/projects/:id` — 상태/날짜/메모 등 업데이트
- [ ] `DELETE /api/projects/:id` — 삭제 (Drive 폴더에 deprecated 추가)
- [ ] `POST /api/projects/:id/suspend` — 중단 처리 (직전 상태 저장)
- [ ] `POST /api/projects/:id/resume` — 중단 재개

### Planning (기획 완료)
- [ ] `POST /api/projects/:id/planning/complete` — 커리큘럼 링크 + 장/강 정보 제출
  - 장 태스크/강 Lecture 자동 생성 + Drive 폴더 자동 생성

### Tasks
- [ ] `PATCH /api/tasks/:id` — 태스크 상태 변경 (PM만 자유롭게, 외부 관계자는 순서 강제)
- [ ] `PATCH /api/tasks/:id/dates` — 일정 드래그 저장

### Lectures
- [ ] `PATCH /api/lectures/:id/review` — 검수 토글 (검수자)
- [ ] `PATCH /api/lectures/:id/approval` — 승인 토글 (PM)
- [ ] `PATCH /api/lectures/:id/lesson-plan` — 교안 링크 등록 (커기매)

### Chapters
- [ ] `POST /api/projects/:id/chapters` — 장 추가 (Drive 폴더 생성 포함)
- [ ] `PATCH /api/projects/:id/chapters/:ch` — 제목/분량 수정
- [ ] `DELETE /api/projects/:id/chapters/:ch` — 장 삭제

### External (연동)
- [ ] `GET /api/slack/channels?q=` — 슬랙 채널 검색
- [ ] `GET /api/slack/channels/:id/members` — 채널 멤버
- [ ] `POST /api/drive/upload-webhook` — Drive Push Notification 수신
