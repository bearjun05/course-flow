## 프로젝트 개요

온라인 강의 제작 프로세스(**기획 → 교안 → 촬영 → 편집·검수 → 완료**)를 관리하는 PM 대시보드.

### 3대 핵심 가치

| 가치 | 설명 |
|---|---|
| **병목 제거** | 단계 간 정보가 구조화되어 전달, 추가 커뮤니케이션 없이 다음 단계 즉시 착수 |
| **일정/품질 관리** | 각 단계의 일정 준수·품질 기준 충족 여부를 시스템으로 관리 |
| **시스템 중심 협업** | 사람이 아닌 시스템이 진행 상태를 추적, 자동 알림으로 정보 격차 해소 |

### 두 가지 진입점

- **에듀옵스** (`/`): 내부 PM + 커기매용 대시보드 (전체 조회·편집)
- **에듀웍스** (`/eduworks`): 외부 튜터/편집자/자막자/검수자용 (본인 배정 프로젝트 한정, 읽기 전용 + 검수자 토글)

역할·권한 세부는 [`policies/정책-초안.md §5`](policies/정책-초안.md) 참고.

## 관리 범위 기준

다음 3가지를 **모두 충족**하는 강의가 관리 대상:

| 분류 | 대상 |
|------|------|
| **사업부** | KDT (내일배움캠프) / KDC (스파르타클럽) / 기타 (신사업) |
| **제작 유형** | 신규 제작 / 부분 리뉴얼 / 전체 리뉴얼 |
| **공정** | 촬영+편집·검수 전 과정 / 편집·검수만 |
