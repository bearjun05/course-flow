1. 프로젝트 상태
    - 강의 제작 프로젝트 전체의 생애주기(Life-cycle)를 관리하는 상태값
        - 기획: 교육운영팀에서 커리큘럼 및 강의 기획을 진행 중인 상태
        - 교안 제작: 완성된 커리큘럼을 기반으로 튜터님이 교안을 제작 중인 상태
        - 촬영/제작: 교안 제작 완료 이후 촬영~편집/검수가 진행되 상태
        - 리뷰: 모든 제작 공정이 완료되어 최종 승인을 위한 검토가 진행 중인 상태
        - 완료: 최종 검수 통과 후 롤아웃(배포) 완료
        - 중단: 완성되지 않았지만 내부 사정에 의해 제작이 일시 중지되거나 취소된 상태

1. 세부 태스크
    - 태스크의 성격에 따라 두 가지 유형으로 분류하여 관리합니다.
    - 공통 태스크
        - 특정 챕터에 귀속되지 않고 프로젝트 전반에 걸쳐 1회성으로 발생하는 과업 (리허설 촬영, 최종 롤아웃 )
        - 상태값
            - 대기: 작업 시작 전
            - 진행: 작업 중
            - 완료: 과업 종료
    - 챕터별 반복 태스크
        - 각 챕터별로 동일하게 반복되는 일련의 표준 공정 세트 (교안제작-촬영-편집-검수)
        - 상태값
            - 대기: 작업 시작 전
            - 진행: 작업 중
            - 리뷰: 작업 완료 후 PM이 확인하는 단계
            - 완료: 과업 종료
