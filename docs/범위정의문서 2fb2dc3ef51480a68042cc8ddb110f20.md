# 범위정의문서

ETA: 2026년 2월 4일
날짜: 2026년 2월 2일

### 1. 관리 기준

- **사업부 기준**
    - KDT / KDC / 신사업
- **제작 유형 기준**
    - 신규 제작 / 부분 리뉴얼 / 전체 리뉴얼
- **제작 공정 기준**
    - 촬영 + 편집·검수 / 편집·검수만 진행

### 2. 사업부별 관리 대상

- **KDT**
    - 내일배움캠프 트랙 내에서 제작되는 모든 강의
    - 관련 부서: KDT 교육운영팀
- **KDC**
    - 스파르타클럽 내에서 단일 판매되는 모든 강의
    - 관련 부서: 교육운영실 재직자/KDC 교육운영 파트
- **신사업**
    - 신사업 영역에서 제작되는 모든 강의 (부업, 시니어 등)
    - 관련 부서: 교육운영실 재직자/KDC 교육운영 파트, B2C사업기획팀, 그로스실 등

### 3. 제작 유형별 관리 대상

🔖

참고 사항 ⇒  버전 표기 **n.m** 

</aside>

- **신규 제작**
    - 커리큘럼 기획부터 촬영, 편집·검수까지 전 과정을 거치는 강의
        - 백오피스: **신규 강의**로 생성
    - 일정 / 품질 / 정산 관리가 모두 필요한 유형
- **부분 리뉴얼**
    - 기존 강의 중 일부 챕터 재촬영 또는 재편집이 필요한 강의
        - 백오피스: 기존 강의 내에서 신규 버전 생성 *(교육과정: n.m → m 변경)*
    - 변경 범위에 따라 관리 항목이 달라짐
        - 유지보수 → 일정 / 품질 관리가 필요한 유형
        - 신규 내용 추가 → 일정 / 품질 / 정산 관리가 모두 필요한 유형
- **전체 리뉴얼**
    - 기존 강의를 전면 재촬영하는 경우
        - 백오피스: 기존 강의 내에서 신규 버전 생성 *(교육과정: n.m → n 변경)*
    - 신규 제작과 유사한 수준의 일정 / 품질 / 정산 관리 필요

### 4. 제작 공정별 관리 대상

- **촬영 + 편집/검수**
    - 튜터 촬영과 함께 편집·검수 공정이 연속적으로 진행되는 강의
        - 일정 종속성이 높아 단계 간 병목 관리가 필요함
- **편집/검수만 진행**
    - 기존 촬영본을 활용하여 편집·검수만 진행하는 강의
    - 본촬영 없이 편집·검수 공정만 존재하는 경우 해당 공정으로 분류함
        - 실시간 강의의 VOD 전환
        - 기존 강의 커리큘럼 수정 *(하나의 강의를 분리하거나, 복수 강의를 통합하는 작업 등)*

### 5. 관리 범위 요약

다음 조건을 **모두 충족하는 강의**를 관리 대상으로 정의함

- 사업부 기준: **KDT / KDC / 신사업** 중 하나
- 제작 유형 기준: **신규 제작 / 부분 리뉴얼 / 전체 리뉴얼** 중 하나
- 제작 공정 기준: **촬영 + 편집/검수 또는 편집/검수만** 포함
