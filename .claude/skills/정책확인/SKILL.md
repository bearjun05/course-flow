---

사용자에게 아직 결정되지 않은 비즈니스 정책들을 보여주고, 하나씩 확인합니다.

1. `docs/policies/` 아래 모든 .md 파일을 읽는다
2. 각 파일에서 `## 미결정 사항` 섹션의 체크박스(`- [ ]`)를 수집
3. 파일별로 그룹화해서 목록을 보여준다:

출력 형식 (예시):
```
아직 정해지지 않은 것들이 있어요. 하나씩 확인해볼까요?

### 상태 관련 (4건)
1. 프로젝트 상태를 이전 단계로 되돌릴 수 있나요?
2. "중단" 상태에서 다시 시작하면 어떤 단계로 돌아가나요?
3. ...

### 검수 관련 (4건)
1. 몇 점 이상이면 합격인가요?
2. ...

어떤 것부터 정하시겠어요?
```

4. 사용자가 답변하면:
   - `docs/policies/` 해당 파일에서 체크박스를 체크하고 답변 내용을 규칙으로 추가
   - 관련 코드에도 반영

기술 용어 없이 쉬운 말로 질문하세요.
